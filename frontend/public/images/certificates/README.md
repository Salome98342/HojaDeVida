# Carpeta de Imágenes de Certificados

Coloca aquí la imagen de tu certificado con el siguiente nombre:

- `fullstack-spring-angular.jpg` o `fullstack-spring-angular.png` - Certificado Full-Stack Empresarial Spring y Angular (DevSeniorCode)

## Formato recomendado:
- Formato: JPG o PNG
- Resolución: 1200x800 píxeles (aproximadamente)
- Tamaño: Menor a 500KB por imagen
- Calidad: Alta definición para que se vean bien los detalles

## Cómo agregar la imagen:
1. Guarda tu certificado escaneado o captura de pantalla del certificado de DevSeniorCode
2. Renómbralo a `fullstack-spring-angular.jpg` (o .png)
3. Cópialo en esta carpeta
4. La imagen aparecerá automáticamente en el modal al hacer clic en el certificado
